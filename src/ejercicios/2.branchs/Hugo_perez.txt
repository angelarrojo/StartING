2.1. Preguntas
¿Qué es un branch?
Es para poder hacer funciones nuevas y corregir errores
¿Por qué pueden ser útiles los branches?
Para trabajar en equipo, probar ideas y organizar cada su trabajo
¿Cómo se crea una branch?
Git branch (nombre que queramos)
¿Cómo se cambia a una branch?
Git checkout (nombre que hayamos puesto)
¿Cómo se elimina una branch?
Git checkout -d nombre que hayamos puesto
¿Cómo se crea una branch y se cambia a ella en un solo paso?
Git checkout -b nombre que hayamos puesto
¿Qué es un merge?
Es para unir dos ramas en una
¿Cómo se realiza un merge?
Git checkout merge nombre que hayamos puesto
¿Qué es un tag?
Es para marcar algo en específico
¿Cómo se crea un tag?
Git tag -a nombre que queramos en el tag -m "breve descripción"
