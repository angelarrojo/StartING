1.1. Preguntas

1. ¿Cómo se inicializa un repositorio local? (que comando se debe ejecutar?)
- Debemos crear primero una carpeta para trabajar todo el proyecto y para ejecutar comandos abrir la terminal de intelij y poner git init
2. ¿Cómo hago para que un directorio deje de ser controlado por git? (que comando se debe ejecutar?)
- Hay que meterse en la terminal y poner el siguiente comando: -rf .git
3. Si agrego un archivo a un directorio que ya está siendo controlado por git, ¿está siendo controlado por git?
- No, porque en ese caso el archivo no controlaria el git, solo lo detectaria, se hace manualmente con git add
4. ¿Qué comando se utiliza para agregar un archivo al repositorio local?
- git add (nombre del archivo)
5. ¿Cómo determino que archivos fueron modificados? (que comando se debe ejecutar?)
- Para ver los archivos que han sido modificados usamos el comando git status que se refiere al estado
6. ¿Qué comando se utiliza para hacer un commit?
- Muy sencillo, simplemente git commit
7. En sus propias palabras, ¿qué es un commit?
- Un commit es el momento que guardamos el proyecto el cual nosotros guardamos todos los cambios que hayamos realzado hasta ese punto junto a un comentario, es como hacer una foto del proyecto en un momento exacto
