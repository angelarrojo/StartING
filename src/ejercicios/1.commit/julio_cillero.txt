1. ¿Cómo se inicializa un repositorio local? (que comando se debe ejecutar?)
Tendriamos que crear una carpeta en la que fueramos a trabajar, despues dentro del proyecto meternos en la terminal (ALT+F12) y escribir git init
2. ¿Cómo hago para que un directorio deje de ser controlado por git? (que comando se debe ejecutar?)
Tendriamos que meternos en la terminal y poner el comando rm -rf .git
3. Si agrego un archivo a un directorio que ya está siendo controlado por git, ¿está siendo controlado por git?
No, el archivo detectaria el git pero no lo controlaria ya que para eso lo tendriamos que poner nosotros con el comando git add
4. ¿Qué comando se utiliza para agregar un archivo al repositorio local?
Para agregar un archivo vamos a tener que poner el comando git add "nombre del archivo"
5. ¿Cómo determino que archivos fueron modificados? (que comando se debe ejecutar?)
Para ver los archivos que han sido modificados vamos a utilizar el comando git status
6. ¿Qué comando se utiliza para hacer un commit?
Para hacer un commit vamos a tener que poner git commit -m "El mensaje que queramos poner"
7. En sus propias palabras, ¿qué es un commit?
Un commit es un guardado que nosotros hacemos junto a un comentario para poder volver a esa version cuando queramos
