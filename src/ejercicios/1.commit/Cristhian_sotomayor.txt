1. ¿Cómo se inicializa un repositorio local? (que comando se debe ejecutar?)
Para inicializar un repositorio debemos poner el comando git init
2. ¿Cómo hago para que un directorio deje de ser controlado por git? (que comando se debe ejecutar?)
Para poder hacerlo debemos borrar la carpeta .git
3. Si agrego un archivo a un directorio que ya está siendo controlado por git, ¿está siendo controlado por git?
git ya detecta el archivo pero tenemos que añdirlo manualmente con git add
4. ¿Qué comando se utiliza para agregar un archivo al repositorio local?
El comando que debemos utilizar es git add nombre_del_archivo
5. ¿Cómo determino que archivos fueron modificados? (que comando se debe ejecutar?)
Para saber que archivos fueron modificados debemo poner el comando git status
6. ¿Qué comando se utiliza para hacer un commit?
El comando que debemos utilizar es git commit -m mensaje
7. En sus propias palabras, ¿qué es un commit?
Un commit es para poder guardar versiones del proyecto git tambien para se puede volver a versiones anterirores por si nos ocurre algun error y tambien se registra un historial de todo lo que ha ido modificando en el proyecto
