Ejercicio 1

1.1. Preguntas

1.¿Cómo se inicializa un repositorio local? (que comando se debe ejecutar?)
Se ejecuta el comando git init.

2.¿Cómo hago para que un directorio deje de ser controlado por git? (que comando se debe ejecutar?)
-rm -rf .git eso es para borrar la carpeta git.

3.Sí agrego un archivo a un directorio que ya está siendo controlado por git, ¿está siendo controlado por git?
-No, solo será controlado por git cuando se usa git add.

4.¿Qué comando se utiliza para agregar un archivo al repositorio local?
-git add <el nombre que queramos>.

5.¿Cómo determino que archivos fueron modificados? (que comando se debe ejecutar?)
-git status nos ayuda a ver los archivos que han cambiado.

6.¿Qué comando se utiliza para hacer un commit?
-git commit -m "el mensaje que queramos"

7.En sus propias palabras, ¿qué es un commit?
-Un commit es para guardar la información que hayamos hecho, y lo hacemos cuando sea necesario.
