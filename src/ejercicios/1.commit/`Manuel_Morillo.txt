 1.1 PREGUNTAS TEORICAS.¿Cómo se inicializa un repositorio local? (que comando se debe ejecutar?)
Tienes  que crear una carpeta para tu proyecto y abres la terminal en esa carpeta y pones el comando git init

 ¿Cómo hago para que un directorio deje de ser controlado por git? (que comando se debe ejecutar?)
Debo de utilizar el siguiente comando git rm --cached -r
. Si agrego un archivo a un directorio que ya está siendo controlado por git, ¿está siendo controlado por git?
No tiene porque ya que para que esre controlado tendrias que poner tu mismo  un comando.

¿Qué comando se utiliza para agregar un archivo al repositorio local?
 Se utilizan dos comandos git add para preparar el archivo y luego git commit para guardarlo permanentemente en el repositorio local

 Cómo determino que archivos fueron modificados? (que comando se debe ejecutar?)
 Con git status ves los archivos organizados por colores y cuales fueron modificados.

 ¿Qué comando se utiliza para hacer un commit?
  se utiliza el comando git commit -m esa -m funcona para poner  la descripcion que quieras.

  En sus propias palabras, ¿qué es un commit?
  Un commit es una descripcion de un proyecto tiene que ser detllaada ya que es muy importante tanto para ti como los demas a la hora de traabajar.
